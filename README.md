## 사용 방법 - macOS 기준

```bash
brew install elixir     # Elixir 설치 - with Homebrew
elixir -v               # Elixir 버전 확인
iex                     # IEx 대화형 쉘
elixir [파일명].exs       # Elixir 파일 실행
```
